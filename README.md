# Lung Cancer Detection Using Deep Learning

## Project Overview

This project was developed as a Bachelor of Engineering (Information Technology) final-year project.

The objective of the system is to detect lung cancer from CT scan images using Deep Learning techniques. The proposed system uses Convolutional Neural Networks (CNN) for image classification and assists in early-stage lung cancer diagnosis.

## Technologies Used

* Java
* JSP
* JavaScript
* MySQL
* Apache Tomcat
* Eclipse IDE
* Deep Learning
* Convolutional Neural Network (CNN)

## Features

* User Registration and Login
* CT Scan Image Upload
* Image Preprocessing
* Feature Extraction
* CNN-Based Classification
* Prediction of Lung Cancer Type

## Project Objectives

* Detect lung cancer from CT scan images.
* Improve diagnosis accuracy using Deep Learning.
* Automate image classification for medical diagnosis support.
